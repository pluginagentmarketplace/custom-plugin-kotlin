# /roles

Browse all 65+ development roles organized by category. Explore role descriptions, responsibilities, and find your next career path.

## Description

Discover all available development roles, their responsibilities, salary ranges, market demand, and learning requirements.

## Usage Examples

```
/roles                      # Show all roles with categories
/roles --search backend    # Search for roles containing "backend"
/roles --category devops   # Show only DevOps-related roles
/roles --level intermediate # Show roles for intermediate developers
/roles --trending          # Show trending/in-demand roles
```
