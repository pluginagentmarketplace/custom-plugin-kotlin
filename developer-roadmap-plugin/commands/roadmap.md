# /roadmap

Explore and select a learning roadmap by development role. Browse 65+ curated learning paths covering all technology domains.

## Description

Choose a development role and get a personalized learning roadmap with structured learning path, resource recommendations, and skill checkpoints.

## Usage Examples

```
/roadmap                    # List all available roles
/roadmap frontend          # Get Frontend Developer roadmap
/roadmap backend           # Get Backend Developer roadmap
/roadmap devops            # Get DevOps Engineer roadmap
/roadmap react             # Get React Developer roadmap
/roadmap kubernetes        # Get Kubernetes Engineer roadmap
/roadmap ai-engineer       # Get AI Engineer roadmap
/roadmap engineering-manager  # Get Engineering Manager roadmap
```

## Roadmap Structure

Each roadmap includes:
1. **Overview** - Role description and responsibilities
2. **Learning Path** - Structured progression from beginner to advanced
3. **Technology Stack** - Tools, frameworks, and languages
4. **Key Skills** - Essential and advanced competencies
5. **Resources** - Documentation, tutorials, courses
6. **Projects** - Hands-on practice projects
7. **Verification Checkpoints** - Skills to master at each level

## Tips

- Start with "Beginner" phase before moving to "Intermediate"
- Build projects at each level
- Join communities and contribute to open source
- Stay updated with latest technologies
- Practice consistently
