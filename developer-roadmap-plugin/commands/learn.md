# /learn

Start a guided learning path with personalized recommendations, resources, and milestones.

## Description

Begin your development journey with structured, step-by-step guidance. Choose your target role, learning pace, and get daily recommendations.

## Usage Examples

```
/learn                           # Start interactive guide
/learn --role frontend          # Learn frontend development
/learn --role backend --advanced # Advanced backend learning
/learn --skill react            # Learn specific skill/technology
/learn --projects              # Get hands-on project recommendations
/learn --resume 35xya2         # Resume previous learning session
```

## Tips for Success

1. **Consistency**: Learn every day, even 30 minutes
2. **Projects**: Build real projects, not just tutorials
3. **Community**: Join communities, ask questions
4. **Practice**: Repeat concepts until mastered
5. **Read Code**: Study open source projects
6. **Build**: Create your own projects
7. **Share**: Write about what you learn
8. **Mentor**: Teach others to solidify knowledge
