# /learn-plugin

Start your custom plugin development journey with guided learning paths, structured curriculum, and hands-on projects.

## Description

Get started with plugin development through interactive lessons, real-world examples, and progressive skill building. Choose your learning pace and focus area.

## Usage Examples

```
/learn-plugin                          # Start interactive guide
/learn-plugin --path kotlin           # Kotlin fundamentals
/learn-plugin --path architecture     # Plugin architecture
/learn-plugin --level advanced        # Skip to advanced topics
/learn-plugin --project               # Jump to projects
/learn-plugin --resume {session-id}   # Resume previous session
```
